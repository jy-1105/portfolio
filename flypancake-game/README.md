# FLY_PANCAKE

## 概要
マウス操作でパワーゲージを調整し、パンケーキをフライパンからフライパンへと飛ばし、最終的に皿に到達させることを目的とした3Dアクションゲームです。  
タイミングに応じて飛距離や軌道が変化するため、精密な操作が求められます。  
さらに、移動する障害物やパンケーキのサイズが変化するアイテムを配置し、高難易度で繰り返し挑戦したくなるゲーム性にしました。

## 開発期間
2024年10月〜2024年12月

## 開発体制
3人チーム開発

## 使用技術
- Unity
- C#

## 担当部分
- マウス入力によるパンケーキの発射処理の実装
- パワーゲージシステムの実装
- パンケーキの衝突判定およびゲームオーバー処理
- UI（ゲージ）の実装
- 効果音およびBGMの実装

## 工夫した点
- プレイヤーがタイミングを意識して操作できるよう、ゲージの増減を繰り返す仕組みを設計しました。  
- プレイヤーが思い通りに飛ばせたと感じられるよう、発射力や加速度を調整しました。  
- 移動する障害物やサイズ変化アイテムを導入し、プレイ中に判断を求めるゲーム設計を行いました。
- 繰り返し挑戦したくなるゲーム体験を目指し、BGMや効果音にもこだわって、プレイ中の緊張感や達成感を高めました。

## 課題と解決
### 課題
ゲームの難易度調整が難しく、パンケーキが着地しやすいバランスを取る必要がありました。

### 解決
パンケーキとフライパンのサイズを調整し、実際にプレイを繰り返しながら最適なバランスを検証しました。  
その結果、難易度を保ちつつも、プレイヤーが繰り返し挑戦できるバランスに改善しました。
