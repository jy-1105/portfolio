# flypancake-game

## 概要
マウス操作でパワーゲージを調整し、パンケーキをフライパンからフライパンへと飛ばし、最終的に皿に到達させることを目的とした3Dアクションゲームです。  
タイミングに応じて飛距離や軌道が変化するため、精密な操作が求められます。  
さらに、移動する障害物やパンケーキのサイズが変化するアイテムを配置し、高難易度かつ挑戦意欲を刺激するゲーム性を実現しました。

## 開発期間
2024年10月〜2024年12月

## 開発体制
4人チーム開発

## 使用技術
- Unity
- C#
- Rigidbody（物理演算）
- UI（Slider）

## 担当部分
- マウス入力によるパンケーキの発射処理の実装
- パワーゲージシステムの実装
- パンケーキの衝突判定およびゲームオーバー処理
- UI（ゲージ）の実装
- 効果音の実装

## 工夫した点
- ゲージの増減を繰り返す仕組みを取り入れ、タイミングによって飛距離が変わるよう設計しました。
- 実際にプレイを繰り返しながら、適切な加速度や発射力を調整し、操作の気持ちよさを追求しました。
- 移動する障害物やサイズ変化アイテムを導入し、単調にならないゲーム設計を行いました。

## 課題と解決
### 課題
ゲームの難易度調整が難しく、パンケーキが着地しやすいバランスを取る必要がありました。

### 解決
パンケーキとフライパンのサイズを調整し、実際にプレイを繰り返しながら最適なバランスを検証しました。  
その結果、難易度を保ちつつも達成可能な設計に改善しました。
