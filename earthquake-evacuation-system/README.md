# earthquake-evacuation-system

## 概要
地震発生時の避難行動を体験・訓練することを目的とした体験型防災システムです。  
Arduinoと距離センサーを用いて、実際に机の下に潜り込む動作を検知し、ゲーム内のキャラクターに反映される仕組みを実装しました。  
地震発生時には机の下に避難し、揺れが収まった後に移動して脱出する流れを体験できます。

## 開発目的
教育用途を想定し、ユーザーが身体を使って避難行動を体験できるインタラクションを通じて、防災意識の向上を目指しました。

## 工夫した点
- 実際に机の下に潜る動作をゲームに反映することで、現実に近い体験ができるよう設計しました。  
- センサーの入力に応じてキャラクターの状態を変化させることで、直感的なインタラクションを実現しました。  
- 視覚・音・動作を組み合わせることで、緊急時の臨場感を高める工夫を行いました。

## 課題と解決
### 課題
距離センサーの値が不安定で、正確な入力判定が難しいという問題がありました。

### 解決
しきい値を設定し、一定条件を満たした場合のみ状態を変化させる仕組みを導入しました。  
これにより、誤検知を抑えつつ安定した入力処理を実現しました。

## 主要スクリプト
- [SerialReader.cs](./scripts/SerialReader.cs)  
  Arduinoとのシリアル通信を行い、センサー入力を取得

- [DistanceSensorReader.cs](./scripts/DistanceSensorReader.cs)  
  距離センサーの値を基に、机の下に入った状態を判定

- [Test_EmergencyManager.cs](./scripts/Test_EmergencyManager.cs)  
  地震発生から避難・終了までのゲーム進行を管理
